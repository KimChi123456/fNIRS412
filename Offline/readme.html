Matlab Software for Sparsity-Assisted Signal Smoothing (SASS)

Version 2 (under construction)

This is a MATLAB software package for the denoising of 1D signals using the SASS approach. In the SASS approach, the signal of interest x(t) is modeled as the sum of a low-pass component f(t) and a sparse order-K derivative component g(t) [x(t) = f(t) + g(t)]. It is assumed that the observed data is corrupted by additive white Gaussian noise (AWGN) w(t) [y(t) = x(t) + w(t)].

Matlab Programs

  • sass.m: SASS algorithm
  • sass2.m: SASS algorithm with zero-locking correction
  • ABfilt.m: Utility function for creating sparse filter matrices

Examples in Matlab

  • Example 1: Discontinuous derivative
  • Example 2: ECG signal denoising
